# Skala Studio Support & Privacy

This repository hosts the public support and privacy policy documents for Skala Studio apps. It exists as a temporary home until a dedicated website is available and is linked from App Store Connect.

## Structure
- `docs/support/` contains support pages by locale, named `support-<locale>.md` (for example, `docs/support/support-en-US.md`).
- `docs/privacy/` contains privacy policies by product and locale (for example, `docs/privacy/unified-calendar-sync/privacy-en-US.md`).
